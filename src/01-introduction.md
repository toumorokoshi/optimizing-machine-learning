# Introduction

Welcome to **Optimizing Machine Learning Models**!

This book is a collection of practical techniques, patterns, and guides for profiling, optimizing, and deploying high-performance machine learning models.

## Interactive Code Guide
All code snippets and benchmarks shown in this book are pulled directly from live, compilable source code files in the `examples/` directory using mdBook's relative file inclusion syntax:
```
\{{#include ../examples/path/to/source.cc}}
```

This guarantees that:
- Every code example builds successfully and works correctly.
- Code blocks in the book never go out of date as the repository evolves.
- You can compile, run, and benchmark any example locally using **Bazel**.
